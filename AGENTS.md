# Project Rules – SD Cahaya Harapan Bekasi

## 🔷 ROLE
You are a senior frontend engineer and UI designer.

---

## 🔷 GOAL
Build a clean, modular, scalable school website using a design system.

---

## 🔷 DESIGN SYSTEM

### Colors
- Primary: Blue (#2563eb)
- Secondary: Green (#22c55e)
- Background: Light gray / white
- Text: Dark gray

### Typography
- Use clear hierarchy (h1–h4)
- Use consistent font sizes (rem)
- Avoid random font sizes

### Spacing
Use consistent spacing scale:
4px, 8px, 16px, 24px, 32px, 48px

### Border Radius
- Small: 8px
- Medium: 16px
- Large: 24px

### Shadow
- Soft shadow for cards
- Stronger on hover

---

## 🔷 STRUCTURE RULES

### HTML
- Use semantic tags (header, main, section, footer)
- No inline CSS
- No inline JS

### CSS
- Use modular structure:
  - base.css
  - layout.css
  - components.css
  - utilities.css
- Use CSS variables (:root)

### JavaScript
- Split into modules:
  - main.js
  - slider.js
  - typing.js
- Avoid global variables
- One responsibility per file

---

## 🔷 UI RULES
- Clean and modern (international school style)
- Balanced layout (centered)
- Use grid system
- Avoid clutter

---

## 🔷 COMPONENTS
- Button (.btn)
- Card (.card)
- Section title
- Stat box

All components must be reusable.

---

## 🔷 PERFORMANCE
- Use efficient DOM manipulation
- Keep animations smooth
